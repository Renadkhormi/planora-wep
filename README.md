# ✈️ Planora — Smart Travel Planner

Planora is a modern travel planning web application designed to help users explore destinations, manage budgets, and generate personalized travel plans easily.

---

## 🌍 Features

- Select country and city  
- Enter travel duration and budget  
- Get smart travel suggestions  
- View:
  - Capital  
  - Currency (with SAR conversion)  
  - Languages  
  - Timezone  
  - Region  
- Budget status analysis  
- Travel vibe recommendation  
- Quick travel advice  
- Top attractions, restaurants, and hotels  
- Suggested day-by-day itinerary  

---

## 🧠 Concept

Planora focuses on simplifying travel planning by combining:

- Clean UI  
- User-friendly experience  
- Smart content generation  

The goal is to help users make better travel decisions quickly without complexity.
